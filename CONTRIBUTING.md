# Contributing to nostr-secprobe

Thanks for your interest in contributing! Please follow these guidelines:

- Fork the repo and create feature branches from `main`.
- Use Go 1.22+ and run `go fmt ./...` and `go test ./...` before submitting.
- Keep the CLI cross-platform and avoid CGO unless behind build tags.
- Add tests for new features and bug fixes.
- For significant changes, open an issue to discuss first.
- By contributing you agree to license your contributions under the MIT License.
